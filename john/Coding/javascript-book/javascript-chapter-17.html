Chapter 17: Changing arrays

.pop - removes an element at the end of an array
.push - adds an elements at the end of an array
.shift - removes an element at the start of an array
.unshift - adds an element at the start of an array
.splice - inserts elements anywhere in the array, and can also remove elements that come after it.
.slice - cuts an element from one array, removing it and copying it to another array
