Using .childNode[] you can target an element that is a child of a parent node

You can target the first child node by using parentNode.firstChild;

You can target the last child node by using parentNode.lastChild;

You can also find the parent node by using document.getElementById and assigning it to a variable

Nodes have siblings, which are other nodes under the same parent node and you can target these the same way you do to parent nodes but use .nextSibling and .previousSibling instead
